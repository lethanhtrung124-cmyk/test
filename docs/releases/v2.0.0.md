# Release v2.0.0

Ngày phát hành: 15/07/2026

## Phạm vi

Baseline tự động hóa chức năng theo Use Case, gồm Web App, Supabase migration, Playwright framework, GitHub Actions, Netlify config và tài liệu vận hành.

## Bằng chứng kiểm thử

Khi môi trường Node được cài đặt, chạy:

```bash
npm install
npm run build
npm run test:e2e
```

Kết quả cần lưu:

- `test-results/junit.xml`
- `test-results/summary.json`
- `playwright-report/`

## Ghi chú triển khai

Phiên bản này có chế độ mock pilot cho frontend. Để vận hành thật cần tạo Supabase project, áp dụng migration/seed, cấu hình RLS thành viên dự án và thiết lập secrets cho GitHub Actions.

## Xác minh tài liệu chính thức

Tại thời điểm tạo baseline, repo đã căn theo các nhóm tài liệu chính thức: Supabase PostgreSQL/RLS/Edge Functions/Storage, Playwright reporters/trace/video, GitHub Actions workflow_dispatch và Netlify build redirects. Trước khi triển khai production, cần xác minh lại phiên bản CLI/dịch vụ đang dùng và cập nhật release note nếu có khác biệt.
